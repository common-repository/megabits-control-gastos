Control de gastos en la propia web.

== Description ==
Sencillo plugin que te permite almacenar las facturas de los gastos en la propia web de tu tienda online.

Pensado para los autónomos y las pequeñas empresas.

Ya que tienes una tienda online en donde guardas las facturas de tus clientes porque no guardar también las facturas de todos aquellos gastos de la tienda en el mismo sitio.

Todo en un mismo sitio, las facturas de teléfono, las del alquiler, las de tus proveedores y todas aquellas que desees ahora estarán disponibles siempre y sin tener carpetas en tu ordenador o engorrosos archivadores.

Sube las facturas en PDF o simplemente como imágenes.

En el calendario se muestran los pagos que tienen una fecha de vencimiento para que veas cuando vencen y pagarlos. Pulsando en cada vencimiento tendrás acceso a consultar la factura.

Podrás filtrar las búsquedas por proveedor, categoría, mes o trimestre del año.

Facilita el trabajo a tu asesoría y descarga un archivo zip con los gastos del mes o del trimestre.

Este no es un plugin de contabilidad pero si ayuda a llevar la contabilidad.

== Installation ==
1. Upload the entire `megabits-control-gastos` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the **Plugins** screen (**Plugins > Installed Plugins**).

== Frequently Asked Questions ==

= ¿Por qué Megabits Control Gasto? =

Basándonos en nuestra propia experiencia resulta más útil tener todo en la propia web. Existen plugins de contabilidad muy completos pero un autónomo no necesita eso, necesita tener sus facturas de gastos a mano y poder pasarlos a la asesoría/gestoría de una manera sencilla, en un simple fichero zip.
Esto es lo que hace este plugin.

= ¿Es Megabits Control Gasto un plugin de contabilidad? =

No. Es un plugin para que puedas guardar tus facturas de gastos en la misma web donde tienes las facturas de tus clientes.
